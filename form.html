Questionario

  1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

    2. R: O atributo "placeholder" em uma tag <input> com o atributo "type" definido como "text" é usado para fornecer um texto de exemplo ou uma dica aos usuários sobre o tipo de informação que é esperado no campo de entrada.

  2. Na tag <input>com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

    3. R:Em<input type="radio">e<input type="checkbox">permite definir opções padrão selecionadas ou caixas

  3. Na tag <input> para que serve o atributo "value"?

    4. R:No caso de botões de rádio ( <input type="radio">) e caixas de seleção ( <input type="checkbox">), o atributo "value" é usado para associar um valor específico a uma escolha.

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    5. R:O atributo checked é usado para indicar quais checkboxes e qual radio button devem estar ligados inicialmente, se necessário.

  5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

    6. R:Um elemento **HTML

  6. Na tag <option> para que serve o atributo "selected"?

    7. R:O elemento HTML select (<select>) representa um controle que apresenta um menu de opções. As opções dentro do menu são representadas pelo elemento <option> , que podem ser agrupados por elementos <optgroup> .

  7. Nas tags de formulário para que serve o atributo "required"?

    8. R:Esse é um atributo booleano usado para indicar que um determinando campo de formulário é obrigatório para o envio do mesmo.

  8. Para que serve a tag <textarea>?

    9. R:O elemento HTML <textarea> representa um controle de edição para uma caixa de texto, útil quando você quer permitir ao usuário informar um texto extenso em formato livre, como um comentário ou formulário de retorno.

  9. Na tag <form> para que serve o atributo "action"?

    10. R:O atributo action define o local (uma URL) em que os dados recolhidos do formulário devem ser enviados.

  10. Na tag <form> para que serve o atributo "method"?

    11. R:O atributo action define o local (uma URL) em que os dados recolhidos do formulário devem ser enviados.

  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?

    12. R:Password: permite que o usuário digite um valor que não é exibido de forma legível, sendo utilizado para a entrada de senhas.

  12. Qual valor deve ser informado no atributo "method" na tag < form > em um formulário que será usado para fazer upload de arquivos?0

    13. R:Se o elemento para um botão ou uma imagem de envio de formulário, este atributo especifica o método HTTP que o navegador deve usar para enviar o formulário.

  13. O que as tags < fieldset > e < legend > produzem?

    14. R:Tanto a tag <fieldset> quanto a tag <legend>podem receber os atributos id , class , title , style , dir , lang e acceskey , e os eventos onclick , ondblclick , onmousedown , onmouseup , onmouseover , onmousemove , onmouseout , onkeypress , onkeydown e onkeyup .
